Mobile Developer

Deskripsi :
  1. Kembangkan Aplikasi Seluler
  2. Tingkatkan Performa Aplikasi Seluler
  3. Berkolaborasi dengan Tim UI/UX untuk mengimplementasikan animasi yang baik
  4. Berkolaborasi dengan Back-End
  5. Memperbaiki Bug
Kriteria :
  1. Pendidikan minimal Sarjana (S1) Sistem Informasi (SI)/ Teknologi Informatika (TI).
  2. Fresh Graduate dapat melamar.
  3. Pengalaman minimal 1 tahun sebagai Mobile Developer,
  4. Lebih disukai Berpengalaman di android java, kotlin, react native, dart (flutter)
  5. Tahu tentang Git
  6. Analitis yang baik.

Game Developer

Deskripsi :
  1. Pengembangan menggunakan Unity 3D untuk Aplikasi Desktop dan Mobile (IOS dan Android)
  2. Berkolaborasi dengan seniman dan desainer tentang persyaratan dan mengubah spesifikasi desain menjadi game fungsional
  3. Teliti, buat, uji, tolok ukur, dan evaluasi teknologi baru
  4. Tingkatkan kinerja, kualitas, dan daya tanggap aplikasi seluler yang ada mengimplementasikan fitur baru dan meningkatkan kinerja
  5. Membuat dan menyempurnakan alat untuk membantu tim dalam pembuatan konten
  6. Mekanika permainan pengujian dan penyempurnaan
  7. Buat struktur Proyek (termasuk ERD)
  8. Tawarkan dan terapkan solusi tepat waktu
Kriteria :
  1. Kandidat harus memiliki setidaknya Gelar Sarjana di Ilmu Komputer/Teknologi Informasi atau setara.
  2. Bahasa yang diperlukan: Bahasa Inggris, Bahasa Indonesia
  3. Setidaknya 2 Tahun pengalaman kerja di bidang terkait diperlukan untuk posisi ini.
  4. Keterampilan pengkodean yang sangat baik menggunakan
  5. Pengalaman dengan mesin game unity (dan Unreal Engine merupakan nilai tambah)
  6. Pengalaman yang dapat dibuktikan dengan mesin game Unity3D
  7. Keterampilan komunikasi yang sangat baik
  8. Keterampilan manajemen waktu yang hebat
  9. Pengetahuan dalam pengembangan game 2D & 3D
  10. Pengalaman di Studio Game

UI/UX Designers

Deskripsi :
  1. Melakukan riset pengguna dan analisis pesaing terkait UI/UX design
  2. Bertanggung jawab untuk memahami arahan proyek dan mengubahnya menjadi UI/UX design yang menarik dan berguna.
  3. Merencanakan dan mengimplementasikan design baru untuk produk digital seperti website atau aplikasi seluler.
  4. Membuat user-centered designs dengan memahami persyaratan yang telah ditentukan oleh bisnis serta feedback dari pengguna.
  5. Membuat user flow, storyboards, wireframe, prototype, dan mockups.
  6. Merancang elemen UI seperti Input Control, Navigational Component, Informational Component, dan Container.
  7. Mengidentifikasi dan memecahkan masalah UI ataupun UX.
  8. Mempresentasikan konsep design kepada pihak internal ataupun klien
Kriteria :
  1. Menguasai Software: Figma, Adobe (Photoshop, Illustrator)
  2. Operating System: Windows
  3. Berpengalaman dalam handle Multiple Project
  4. Mampu bekerja dibawah tekanan dan target/deadline
  5. Kreatif, Detail dan dapat berkomunikasi dengan baik
